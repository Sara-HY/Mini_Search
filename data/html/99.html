我现在是自认为有点技术，技术到什么程序，我只能用一个作品来展示（我个人开发的专用系统的一个栏目 图片相册系统 也是技术要求相对较高的 ）。
我投了好多简历，基本没消息。 有消息的时候，面试的一般都是人事部的女的，她们不懂技术只和你谈一些其他的东西。 还有有的公司完全不鸟我的作品什么的 ，只要有经验的，达不到条件更本不看你。 当然我现在面试还不是非常多，但基本都遇到这个情况。

我学历就是大专的， 但公司一般都需要几年经验的。我确实对编程很感兴趣，家里去年有点事，不能出去找工作，在家里一年多的时间学习php 和javascript。一天10来个小时 除了帮家里的忙 ，基本都是学习和打造自己的专用系统。也敢于挑战尝试一些新技术，看到一些大网站的特效新功能，我有很大的兴趣去分析怎么实现的，然后敢于挑战复杂的实现。但找工作的时候我把希望寄托在自己搞的开发作品上，但几乎无效果。 我该怎么办。 各位给我指点下出路，谢谢。

-------------------------------------------------------------------
地址：http://yangdx.6t.dk/view/tpsc.html   （免费空间 速度可能很慢 又是相册系统 ）
这个是我那个个人专用系统的相册栏目 ，开发一多半了，单独拿出来展示，没有做居中对齐，已经去掉登陆验证了。在chrome下开发调试 ，请用chromeium的浏览器观看，暂时没精力去完善其他的浏览器。   参照了百度相册 qq空间相册  facebook的相册  相对于他们来说还是算非常粗糙了。

要说明的是 所有东西都是我自己搞的 div+css    php javascript   ajax 等等 ，有的很难的功能 我也是参考别人的实现思路或者代码，然后按照自己的思路写的。

批量上传功能也是自己写的  flash+flash actionscript3.0+ javascript  能生成缩略图 自己想达到百度相册批量上传的那效果，但现在还差很多。
